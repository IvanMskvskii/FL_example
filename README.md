# FL_example
Симуляция федеративного обучения.
В рамках задачи было проведено федеративное обучение для 5 клиентов, агрегация весов и сравнение итоговой модели с централизованной моделью на отложенной выборке. 
Обе модели имеют одинаковую архитектуру. 

В качестве данных использовался учебный датасет MNIST database. Для федеративного обучения использовались библиотеки TensorFlow и Flower.

В результате федеративное и централизованное обучение прошли успешно. Централизованная модель показала чуть более высокое качество accuracy, чем федеративная (0.9813 и 0.9798 соответственно).
